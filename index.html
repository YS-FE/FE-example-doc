使用

添加发布

  1. 修改SUMMARY.md 添加章节
  2. gitbook build 编译
  3. cp -a _book/* .
  4. gitbook serve 本地查看
  5. git push origin master

一步到位

  • 直接运行脚本(需要可执行权限) bash ./publish.sh

安装插件

  1. 修改 book.json文件
  2. gitbook install 安装
